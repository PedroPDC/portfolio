Integração completa com Power Apps, MySQL e Power BI para Análise e Controle de Produção.
Otimize a gestão industrial com dados! Este projeto conecta Power Apps, MySQL e Power BI para monitorar e analisar KPIs essenciais, como OEE, produtividade e qualidade, proporcionando uma visão clara e em tempo real da produção.
Detalhes do Projeto
Este projeto foi desenvolvido a partir de uma integração completa entre Excel, Power Apps, MySQL e Power BI para análise e controle de produção, buscando otimizar o gerenciamento de dados industriais. Criado para atender à necessidade de empresas que demandam um ciclo completo para lançamento, edição, visualização e exclusão de registros em tempo real, o sistema reduz erros operacionais e elimina o uso de papel no registro de dados produtivos. A solução inclui um aplicativo em Power Apps, permitindo que operadores cadastrem, editem e deletem informações sobre operadores, produtos e ocorrências de manutenção, além de registrar produção e ordens de produção. O aplicativo também possui um botão de acesso direto ao dashboard para acompanhamento dos dados.
Os dados são armazenados em um banco de dados MySQL, garantindo eficiência e segurança no processamento de grandes volumes de informações. Esse banco foi conectado ao Power BI para a criação de um dashboard interativo, onde foram definidos os principais indicadores de desempenho e a melhor disposição dos elementos visuais. O design do relatório foi elaborado no Figma, seguindo um conceito limpo e minimalista, priorizando clareza e facilidade na interpretação dos dados. Por fim, utilizei um recurso nativo do Power BI para garantir que o painel seja atualizado automaticamente, promovendo agilidade e precisão nas análises, facilitando a tomada de decisões estratégicas.
